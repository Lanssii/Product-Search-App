# Product Search App 

A vanilla JavaScript application built to practice and demonstrate core JS concepts. This project focuses on data manipulation, DOM rendering, and event handling.

## Features
- **Live Search:** Filter products in real-time by name, category, or price.
- **Dynamic Sorting:** Toggle between "Price: Low to High" and "Price: High to Low" views.
- **UI Rendering:** Clean and responsive product cards generated entirely through JavaScript.

## Tech Stack & Concepts Used
- **HTML5 & CSS3**
- **JavaScript (ES6+):**
  - **DOM Manipulation:** Using `createElement`, `appendChild`, and `querySelector`.
  - **Array Methods:** Implementing `.filter()`, `.sort()`, and `.forEach()`.
  - **Event Listeners:** Handling `input` for searching and `click` for sorting.
  - **Logic:** Case-insensitive search using `.toLowerCase()` and immutable sorting with the **Spread Operator** `[...]`.

## Learning Objectives
The goal of this project was to combine multiple JavaScript concepts into a functional tool. I focused on:
1. Connecting a data array (objects) to the User Interface.
2. Managing application state when sorting and filtering simultaneously.
3. Writing clean, reusable functions like `renderProducts()`.

## How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
